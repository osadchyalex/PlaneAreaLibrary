Выполненое тестовое задание.

Задание:
Напишите на C# библиотеку для поставки внешним клиентам, которая умеет вычислять площадь круга по радиусу и треугольника по трем сторонам.

Дополнительно:
	- Проведены Юнит-тесты.
	- Реализована проверка на то, является ли треугольник прямоугольным.


Библиотека поддерживает классы:

	"Circle" - для круга
		Включены следующие методы:
			"Area" - Вычисление площади круга по радиусу

	"Triangle" - для треугольника
		Включены следующие методы:
			"Area" - Вычисление площади треугольника по его сторонам
			"Right" - Проверка является ли треугольник прямоугольным
